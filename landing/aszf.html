2. A szolgáltatás leírása

A MiniCRM MCP egy szoftver-szolgáltatás (SaaS), amely az MCP (Model Context Protocol) szabványon keresztül lehetővé teszi a MiniCRM rendszer és a Claude AI (Anthropic) összekötését.

A Szolgáltatás az alábbi funkciókat biztosítja:

  • MiniCRM adatok lekérdezése és kezelése Claude AI felületen keresztül
  • Projektek, kontaktok, teendők és egyéb MiniCRM entitások olvasása, létrehozása és módosítása természetes nyelvű utasításokkal
  • Automatizált munkafolyamatok támogatása AI segítségével

A Szolgáltatás a Felhasználó saját MiniCRM fiókjában tárolt adatokhoz fér hozzá a Felhasználó által megadott API hitelesítő adatok (System ID és API kulcs) alapján. A Szolgáltató nem tárol és nem dolgoz fel MiniCRM adatokat a Felhasználó kifejezett utasításán túlmenően.

3. Regisztráció és licenckulcs

A Szolgáltatás igénybevételéhez a Felhasználónak az alábbi adatokra van szüksége:

  1. Licenckulcs: A Szolgáltató által kiállított, a hozzáférést biztosító egyedi azonosító.
  2. MiniCRM System ID: A Felhasználó saját MiniCRM rendszerének azonosítója.
  3. MiniCRM API kulcs: A Felhasználó saját MiniCRM API hozzáférési kulcsa.

A Felhasználó köteles a licenckulcsot és az API hitelesítő adatokat bizalmasan kezelni, harmadik félnek nem átadni. A licenckulcs egyetlen személy által használható, annak továbbadása vagy megosztása tilos.

A Szolgáltató fenntartja a jogot, hogy a jogosulatlanul használt licenckulcsokat azonnali hatállyal felfüggeszsze.

Az előfizetés feltételei:

7. Felelősségkorlátozás

A Szolgáltató az alábbi esetekben nem vállal felelősséget:

  • Felhasználói utasítások következménye: A Szolgáltatás a Felhasználó által megadott utasításokat hajtja végre. A Szolgáltató nem felelős a Felhasználó által kezdeményezett adatmódosítások, törlések vagy egyéb műveletek következményeiért.
  • MiniCRM API változások: A MiniCRM API a MiniCRM Zrt. terméke, amelynek működésére, elérhetőségére és változásaira a Szolgáltatónak nincs ráhatása. Az API változásaiból eredő átmeneti szolgáltatás-kiesést vagy funkcióvesztést a Szolgáltató nem tudja garantálni, de törekszik az igazodásra az új API verziókhoz.
  • Harmadik félű szolgáltatások: A Claude AI (Anthropic) és a MiniCRM rendszer elérhetősége, működésbiztonsága és adatkezelése a saját szolgáltatóik felelőssége.
  • Vis maior: Természeti katasztrófa, háború, kormányintézkedés, kibertámadás vagy egyéb, a Szolgáltató érdekkörén kívül eső események által okozott szolgáltatáskiesés.
  • Adatvesztés: A Felhasználó felelős a saját MiniCRM adatainak mentéséről és biztonságáról. A Szolgáltató nem vállal felelősséget a Felhasználó CRM rendszerében bekövetkező adatvesztésért.

A Szolgáltató összeszített kártérítési felelőssége nem haladhatja meg a Felhasználó által az utolsó 12 hónapban ténylegesen megfizetett szolgáltatási díjat.
