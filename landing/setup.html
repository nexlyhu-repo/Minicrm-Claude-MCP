Beállítási útmutató

Kösd össze a MiniCRM rendszeredet a Claude AI-val 5 egyszerű lépésben.

Kb. 2 perc
Előfeltételek - mielőtt elkezded

Claude AI fiók

Regisztrálj a claude.ai oldalon, ha még nincs fiókod. Pro vagy Team előfizetés ajánlott.

REST API modul

A MiniCRM-ben aktív REST API + XML szinkronizáció modul szükséges. Nélküle nem férsz hozzá az API kulcshoz.

MiniCRM hozzáférés

Szükséged lesz a System ID-dra (címsorból) és az API kulcsodra (Beállítások menüből).

MiniCRM MCP licenckulcs

A nexly.hu-ról igényelhető. Formátum: lic_xxxxxxx

REST API modul kötelező! A MiniCRM-ben a Beállítások → Előfizetés menüpontban ellenőrizheted. A "Haladó integrációs csomag" alatt található "REST API + XML szinkronizáció" modult kell aktiválni (240 €/év). E nélkül nem férsz hozzá az API kulcshoz!
1

MCP szerver URL másolása

Másold ki az alábbi URL-t. Erre lesz szükséged a következő lépésben, amikor a Claude AI-ban hozzáadod a MiniCRM csatlakozót.

https://minicrmmcp.nexly.digital/mcp
Ezt az URL-t kell majd beillesztened a Claude AI beállításaiba. Tartsd vágólapon!
2

Connector hozzáadása a Claude AI-ban

Nyisd meg az alábbi gombot, vagy navigálj manuálisan: Claude AI → Settings → Connectors → alul görgess le → Add custom connector.

Megnyitás a Claude AI-ban →

A megjelenő ablakban:

claude.ai/settings/connectors

Add custom connector

Egyedi MCP szerver hozzáadása
https://minicrmmcp.nexly.digital/mcp
↑ Ide illeszd be az 1. lépésben kimásolt URL-t
miniCRM
↑ Tetszőleges név, pl. "miniCRM"
Add connector
Ha sikeres, a miniCRM megjelenik a Connectors listában "CUSTOM" címkével.
3

Connect gombra kattintás

A Connectors listában keresd meg a "miniCRM" sort és kattints a mellette lévő Connect gombra.

A Connect gomb egy felugró bejelentkezési ablakot fog megnyitni (következő lépés).
4

Adatok megadása

A felugró ablakban add meg a három adatot: licenckulcs, MiniCRM System ID és API kulcs.

Hol találom a System ID-t?

Lépj be a MiniCRM-be és nézd meg a böngésző címsorát:

https://r3.minicrm.hu/53832/...

A szám az "r3.minicrm.hu/" után — ez a te System ID-d.

Hol találom az API kulcsot?

A MiniCRM-ben navigálj ide:

Beállítások → Rendszer → API kulcs

Ha nem látod az API kulcs menüt, ellenőrizd, hogy aktív-e a REST API modul az előfizetésedben!

5

Kész! Kezdd el használni

A csatlakozás után menj vissza a claude.ai főoldalra és írj be bármit a MiniCRM-mel kapcsolatban. A Claude automatikusan használja a MiniCRM eszközöket.

Példa utasítások
"Listázd ki az összes nyitott projektet az Értékesítés modulban."
"Keresd meg Kiss János kontaktot és mutasd meg az adatait."
"Hozz létre egy új teendőt a #12345 projekthez: holnap fél 10-re hívjam vissza."
A Claude AI mostantól hozzáfér a MiniCRM rendszeredhez. 54 eszköz áll rendelkezésre: kontaktok, projektek, teendők, számlák, és még sok más.

Gyakran ismételt kérdések

Nem látom az API kulcsot a MiniCRM-ben. Mit tegyek?
Az API kulcs csak akkor érhető el, ha aktív a REST API + XML szinkronizáció modul. Ellenőrizd a MiniCRM-ben: Beállítások → Előfizetés. Ha nincs aktiválva, ott tudod megrendelni (a "Haladó integrációs csomag" részén belül, 240 €/év).
Nem jelenik meg a miniCRM a Connectors listában.
Ellenőrizd, hogy pontosan ezt az URL-t adtad-e meg: https://minicrmmcp.nexly.digital/mcp. Ha a "Connectors have moved to Customize" üzenet jelenik meg, kattints a "Go to Customize" gombra és ott add hozzá az egyedi csatlakozót.
A "Csatlakozás" gomb után hibaüzenet jön.
Ellenőrizd a három megadott adatot:
• Licenckulcs: lic_ kezdetű, a Nexly-től kapott kulcs.
• System ID: szám, pl. 53832 (a címsorból).
• API kulcs: a MiniCRM Beállítások → Rendszer → API kulcs menüben található.

Ha minden adat helyes és még mindig nem működik, írj nekünk: info@nexly.digital
Mennyibe kerül a MiniCRM MCP?
14 napos ingyenes próba, utána havi előfizetés. Részletekért látogass el a nexly.hu oldalra, vagy kérj ingyenes bemutatót a főoldalunkon.
Biztonságos ez? Ki fér hozzá az adataimhoz?
A MiniCRM MCP szerver nem tárol adatot. Minden kérést a Claude AI küld a MiniCRM API-nak a te hitelesítő adataiddal, közvetlenül. Az API kulcsod titkosítva van tárolva. Részletekért lásd az adatkezelési szabályzatot.

Segítség kell a beállításhoz?

Írj nekünk és segítünk beállítani, vagy kérj ingyenes bemutatót ahol végigvezetünk a folyamaton.
